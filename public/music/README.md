# 背景音乐

将你想用作背景音乐的音频文件放入此文件夹，并重命名为 `bg.mp3`。

支持格式：MP3（推荐）、Ogg Vorbis。

## 使用方式

1. 准备一首你喜欢的音频文件（建议轻柔、氛围类音乐）。
2. 将文件重命名为 `bg.mp3` 并放入此文件夹。
3. 重新构建并部署项目。
4. 在游戏右侧面板点击「音乐」开关即可播放。

## 音量与格式

- 默认音量为 35%，可在 `src/utils/music.ts` 中调整 `audio.volume`。
- 如需使用其他格式或文件名，请同步修改 `src/utils/music.ts` 中的路径。
